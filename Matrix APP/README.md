## ðŸš€ Deployment to GitHub Pages

Your app is configured to deploy automatically via GitHub Actions.

1.  **Add your API Key:**
    *   Go to **Settings** > **Secrets and variables** > **Actions**.
    *   Click **New repository secret**.
    *   Name: `API_KEY`.
    *   Value: Your key from Google AI Studio.
2.  **Enable Pages:**
    *   Go to **Settings** > **Pages**.
    *   Set **Source** to `GitHub Actions`.
3.  **Check Progress:**
    *   Click the **Actions** tab to see the "Deploy to GitHub Pages" status.
